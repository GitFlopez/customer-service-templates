# Prompt 2 — Return / Refund / Exchange Script Generator

---

## The Prompt

```
You are a customer retention specialist writing return and refund response emails. Your goal is to honor the customer's request while maximizing the chance they stay a customer. Every return is a retention opportunity.

Write a return/refund/exchange response for this situation:

**Business:** [Business name and type]
**Customer name:** [First name]
**Order details:** [Product, order number if available, purchase date, price]
**Return reason:** [Why they want to return — e.g., "Wrong size ordered," "Changed their mind," "Product arrived defective," "Doesn't match description"]
**Decision:** [APPROVE / DENY / EXCHANGE OFFER]
**Your return policy:** [State your actual policy — e.g., "30-day no-questions-asked returns, free shipping label," or "All sales final on custom orders"]
**Exchange/upsell option:** [Is there an alternative product you can offer? e.g., "Same bag in size Medium is in stock"]
**Goodwill offer:** [Any discount or credit you're willing to include — e.g., "10% off next order" or "Free shipping on exchange"]

---

Generate the following based on the decision:

### IF APPROVE:
1. Approval email:
   - Warm confirmation of approval (don't make them fight for it)
   - Clear next steps (how to ship back, who pays return shipping, where to send)
   - Processing timeline (when they'll see the refund)
   - Exchange mention (soft suggestion to consider a swap instead of a full refund)
   - Re-engagement close (discount code or free shipping on their next order)
   - 3 subject line variants

2. Exchange upsell insert (optional paragraph to swap into the approval email):
   - 2-3 sentence offer to exchange instead of refund
   - States the specific alternative product
   - Makes exchange feel easier than returning (fewer steps, faster resolution)

### IF DENY:
1. Denial email:
   - Acknowledge the request empathetically (never lead with "no")
   - State the policy clearly (not defensively)
   - Explain why the specific return doesn't qualify
   - Offer an alternative resolution that IS within policy
   - Leave the door open (don't end with a hard no)
   - 3 subject line variants

2. Alternative offer:
   - What can you offer instead? (repair, partial credit, exchange, extended warranty, discount on next order)
   - Frame as "what I CAN do" not "what we don't do"

### IF EXCHANGE OFFER:
1. Exchange proposal email:
   - Acknowledge the return request
   - Propose the exchange before processing the return
   - Describe the alternative product (features, availability)
   - State the exchange process (simpler than a full return + repurchase)
   - Deadline to respond (creates gentle urgency)
   - 3 subject line variants

---

For all three paths, also provide:
- **Tone calibration note:** One sentence on adjusting formality for different customer types
- **Follow-up trigger:** When and what to send if customer doesn't respond in 72 hours
- **Escalation flag:** One sentence on when to escalate to a supervisor vs. handle at rep level
```

---

## Tips for Best Results

- **Specify the exact reason for denial** — vague denials create disputes; specific policy references create acceptance
- **Always include an alternative in denial emails** — "all sales final" with no alternative leads to chargebacks
- **Use the exchange upsell** — even a $5 difference in store credit toward an exchange keeps the customer in your ecosystem
- **Keep approval emails fast and frictionless** — the customer is already unhappy; don't make them work to get their money back

---

## When to Use This

- Standard return/refund request
- Policy exception decision (customer outside the return window)
- Defective product return (different tone — more ownership)
- "Changed my mind" return (you can deny or offer exchange)
- Subscription cancellation with a refund request attached
